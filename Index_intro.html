Knockout e como ele funciona

O Knockout tem uma estrutura MVVM (Model-View-View Model) é um padrão
para a construção de interfaces com o usuário.

  • Modelo: São os dados armazenados dentro do programa, são indenpendetes da interface do usuário
  • Modelo de visualização: É como o os códigos interagem com o usuário. Se acontecer uma implementação de login o modelo de visualização seria um login
  • Visualização: Como o nome já diz, é onde o usuário irá ter interação com o aplicativo
